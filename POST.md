# 用 Markdown 写简历，我把整个编辑器做成了开源项目

> 告别 Word 排版烦恼，一款所见即所得的 Markdown 简历编辑器。

## 为什么要做这个项目？

写简历这件事，想必每个人都有过类似的经历：

- **用 Word 排版**：调格式比写内容花的时间还长，对齐靠空格、换行靠回车，换个电脑打开格式就变了
- **用在线简历平台**：模板千篇一律，数据存在别人服务器上，无法导出、无法离线编辑
- **用 Markdown 编辑器**（如 Typora）：写内容确实方便，但没有简历专属的 A4 预览，导出 PDF 时排版经常对不上

这些痛点困扰了我很久。于是我想到一个方案：

> **用 Markdown 专注写内容，用模板控制外观，实时预览 A4 纸效果，一键导出 PDF。**

这就是 **ShowCV** 的由来——一个开源的、所见即所得的 Markdown 简历编辑器。免费使用，无需注册，所有数据完全存在你自己的浏览器里。

## ShowCV 是什么？

ShowCV 是一个基于 Markdown 的在线简历编辑器，核心理念是**内容与样式分离**：

- 你只需要用 Markdown 写简历内容
- 通过模板和设置面板控制外观（字体、颜色、间距、布局等）
- 右侧实时预览 A4 纸效果，所见即所得
- 纯前端实现，基于 React + TypeScript + Vite，无需后端

## 功能详解

### 实时预览：所见即所得

ShowCV 的核心体验是左右双栏布局：左侧是 Markdown 编辑器，右侧是 A4 尺寸的实时预览。

编辑器基于 CodeMirror 6 打造，支持完整的语法高亮、自动补全、搜索替换和撤销/重做。你在左侧敲下的每一行 Markdown，都会立即在右侧的 A4 纸上渲染出来。

预览支持两种模式：
- **平铺模式**：内容连续滚动，适合编辑时快速浏览
- **分页模式**：按 A4 纸分页显示，所见即所得，直接对应打印效果

### 四套精心设计的模板

目前提供了 4 套风格各异的简历模板，覆盖主流求职场景：

| 模板 | 风格 | 适合岗位 |
|------|------|---------|
| T1 经典 | 简约大方，稳重得体 | 传统行业、国企 |
| T2 现代 | 简洁精致，专业感强 | 互联网、科技公司 |
| T3 创意 | 左侧边框设计，个性鲜明 | 设计、创意、传媒岗位 |
| T4 活力 | 标签式彩色标题，年轻时尚 | 互联网、新兴行业 |

切换模板即时生效，你之前调好的主题色和字体会自动保留，只需要重新适配间距即可。模板选择面板中的缩略图是实时渲染的——它们会随着你的编辑内容实时更新，而不是静态截图。

### 丰富的样式定制

右侧设置面板提供了精细的样式控制，让你不必妥协于模板的默认样式：

- **主题色**：8 种精选预设色 + 自定义取色器，一键切换整体风格
- **字体**：5 种字体预设——系统默认、苹方、思源黑体、微软雅黑、Times New Roman，投递外企岗位时可以快速切换到衬线字体
- **字号 & 行高**：H1 / H2 / H3 / 正文四级字号独立调节，行高支持精细控制
- **间距**：页面边距、H2 上下间距、H3 上下间距均可独立设置
- **头像**：上传头像后可调节显示尺寸（40-120px）和圆角（0-50%）
- **布局**：头部区域支持左对齐、居中、右对齐三种布局

所有修改实时生效，不需要手动刷新或重新渲染。

### `||` 双栏语法：简历排版利手

这是 ShowCV 最实用的一个创新——`||` 双栏布局语法。

在简历中，我们经常需要让标题和日期分居左右两侧，比如：

```markdown
### 字节跳动 || 2021.07 - 至今
```

渲染效果：

> **字节跳动** ····················· **2021.07 - 至今**

`||` 左侧的内容左对齐，右侧的内容右对齐。这个语法适用于 H2、H3 和段落三种元素，编辑器中 `||` 会自动高亮显示，让你一眼就能区分双栏布局。

相比用 Markdown 表格语法实现同样的效果，`||` 语法更简洁直观；相比直接写 HTML，它更符合 Markdown 的书写习惯。

### 一键导出：PDF + 图片

写好简历后，ShowCV 提供两种导出方式：

- **PDF 导出**：基于浏览器打印功能，精确 A4 排版，打印样式与预览完全一致
- **复制为图片**：一键将简历复制为高清 PNG 图片（2x 分辨率），方便粘贴到聊天窗口或邮件中

导出效果和你在预览中看到的一模一样，不用担心"导出后格式变了"的问题。

### 无服务器分享

ShowCV 的分享功能不需要任何后端服务。点击分享按钮后，你的简历数据（包括内容、模板选择和所有样式配置）会被压缩编码到 URL 中。

把链接发给朋友，对方打开后 ShowCV 会自动解析并还原完整简历——内容、模板、样式一个不少。这个方案的灵感来自 Vue SFC Playground。

### 多简历管理

侧边栏支持管理多份简历，常见使用场景：

- 中文简历 / 英文简历各一份
- 针对不同岗位准备不同版本
- 旧版简历存档备份

支持新建、复制、重命名（双击标签页即可编辑）和删除操作，所有简历数据独立存储在浏览器本地。侧边栏底部还有一个存储用量进度条，实时显示 localStorage 的使用情况。

## 技术实现简述

ShowCV 是一个纯前端项目，技术栈为 React 19 + TypeScript + Vite + Tailwind CSS v4 + CodeMirror 6 + Zustand。这里简要分享几个技术亮点：

**自定义 Markdown 扩展**：通过自定义 remark 插件（`remarkGroupSection`）对 Markdown AST 进行三级重构——H1 提取为 `<header>`，H2/H3 及其内容包裹为嵌套 `<section>`。配合 `PipeSplit` 组件解析 `||` 语法实现双栏布局，让模板系统拥有完整的语义化 DOM 结构。

**段落级分页算法**：分页模式没有使用 CSS `page-break`，而是实现了一套基于 DOM 测量的贪心分页算法——通过 `getBoundingClientRect` 测量每个 section 的实际位置和高度，按段落粒度分配页面，避免内容在页面中间断裂。

**零后端架构**：所有数据通过 Zustand + localStorage 持久化，分享功能基于 fflate zlib 压缩 + URL Hash 编码实现。整个应用无需任何后端服务，可以部署到任何静态托管平台。

如果你对技术细节感兴趣，欢迎直接查看源码，也欢迎提 Issue 和 PR。

### 在线体验

### 本地开发

```bash
git clone https://github.com/showlotus/showcv.git
cd showcv
pnpm install
pnpm dev
```

### 快速上手三步走

1. **选模板**：在右侧设置面板选择一套喜欢的模板风格
2. **写内容**：在左侧编辑器用 Markdown 编写简历
