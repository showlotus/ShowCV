# 用 Markdown 写简历，我把整个编辑器做成了开源项目

> 告别排版烦恼，一款所见即所得的 Markdown 简历编辑器。

## 为什么要做这个项目？

之前在某个在线简历平台上写好了简历，结果导出 PDF 的时候发现要充值会员。嘶～ 于是就自己用 Markdown 写了一份然后借助 AI 调了调样式，勉强能用了，但是样式调整起来很繁琐。

后来就想着自己做一个：用 Markdown 写内容，实时预览效果，调好了直接导出，不收费、不注册、不留数据。折腾了几天，就有了 **ShowCV**。

免费使用，无需注册，所有数据存在你自己的浏览器里。

## ShowCV 是什么？

一个基于 Markdown 的在线简历编辑器，思路很简单：**内容和样式分开管**。

- 用 Markdown 写简历内容。
- 通过模板和设置面板调外观（字体、颜色、间距、布局等）。
- 右侧实时预览 A4 纸效果，所见即所得。
- 纯前端实现，React + TypeScript + Vite，不依赖后端。

## 几个比较好用的功能

- **左右双栏编辑**：左边写 Markdown，右边实时渲染 A4 效果。支持平铺和分页两种预览模式，分页模式和打印出来的效果一致。

- **四套模板，切换不丢配置**：目前有 4 套风格不同的简历模板。切换后主题色和字体会自动保留，只需要重新适配间距。模板选择面板里的缩略图会跟着编辑内容实时更新，不是静态截图。

- **`||` 双栏语法**：这个是我自己用下来觉得最顺手的功能。简历里经常要标题在左、日期在右，比如：

  ```markdown
  ### 某公司 || 2021.07 - 至今
  ```

  渲染效果：**某公司** ····················· **2021.07 - 至今**

  `||` 左边左对齐，右边右对齐，H2、H3 和段落都能用，编辑器里会自动高亮。

- **样式可以细调**：右侧设置面板可以调主题色（8 种预设 + 自定义取色器）、字体（5 种预设，投外企岗位可以切衬线体）、字号行高、间距、头像大小圆角、头部布局对齐方式。改完立刻生效。

- **导出 PDF 和图片**：PDF 基于浏览器打印功能，A4 排版和预览一致；也可以一键复制成 2x 分辨率的 PNG，方便粘贴到聊天窗口或邮件里。

- **分享不需要服务器**：简历数据压缩编码到 URL 里，发链接给别人就能还原完整简历（头像除外，需要对方重新上传）。

- **多份简历管理**：侧边栏可以管理多份简历，比如中英文各一份，或针对不同岗位准备不同版本。支持新建、复制、重命名（双击标签页就能改）和删除。
